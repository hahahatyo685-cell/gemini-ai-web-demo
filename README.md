# Gemini AI Web Demo

Aplikasi web sederhana untuk mendemonstrasikan integrasi Google Gemini AI menggunakan Node.js dan ExpressJS.

## Fitur
- Input prompt teks melalui web
- Respon AI dari Gemini 1.5 Flash
- REST API menggunakan ExpressJS

## Teknologi
- Node.js
- ExpressJS
- Google Gemini API

## Cara Menjalankan
1. npm install
2. Buat file .env dan isi GEMINI_API_KEY
3. node server.js
